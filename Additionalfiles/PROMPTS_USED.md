# Prompts Used

## 1. Document Embeddings
- "Split uploaded text into chunks of 500 characters for embeddings."
- "Generate embeddings for each text chunk using SentenceTransformers all-MiniLM-L6-v2."

## 2. Question Answering
- "Given a user question and a document chunk, generate a human-readable answer."
- "Use GPT-2 or DistilGPT-2 to answer based on the context of the retrieved chunk."

## 3. Frontend
- "Create a minimal but clean frontend with a file upload form and question input."
- "Add linear gradient background and animated buttons."
- "Show answer below the question after submission."

## 4. General Project
- "Explain how to connect ChromaDB with embeddings for a Private Knowledge Q&A app."
- "Provide steps to make the project GitHub-ready with README, ABOUTME, AI_NOTES, and PROMPTS_USED."
